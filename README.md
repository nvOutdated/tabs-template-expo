# Tabs Component App

A modern React Native application built with Expo, featuring advanced UI components and animations.

## Features

- 🎨 Modern UI with TailwindCSS and NativeWind
- 🔄 Smooth animations using React Native Reanimated
- 👆 Advanced gesture handling with React Native Gesture Handler
- 📱 Cross-platform support (iOS, Android, Web)
- 🎯 TypeScript support
- 📊 Data visualization with React Native Gifted Charts
- 📸 Camera and media library integration
- 📍 Location services
- 🔒 Secure storage capabilities
- 🌐 WebRTC support
- 📱 Bottom sheet and modal interactions
- 🎭 Custom animations with @legendapp/motion

## Tech Stack

### Core Technologies
- React Native 0.79.2
- Expo SDK 53
- TypeScript
- TailwindCSS
- NativeWind

### UI & Animation
- React Native Reanimated
- React Native Gesture Handler
- @legendapp/motion

- React Native Gifted Charts
- React Native Carousel

### Navigation
- Expo Router
- React Navigation
- Bottom Tabs
- Material Top Tabs

### Storage & State Management
- AsyncStorage
- Zustand
- Expo Secure Store

### Media & Hardware
- Expo Camera
- Expo Image Picker
- Expo Media Library
- Expo Location
- React Native WebRTC

## Getting Started

### Prerequisites
- Node.js
- npm or yarn
- Expo CLI
- iOS Simulator (for iOS development)
- Android Studio (for Android development)

### Installation

1. Clone the repository
```bash
git clone [repository-url]
cd tabs-component-app
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Start the development server
```bash
npm start
# or
yarn start
```

### Available Scripts

- `npm start` - Start the Expo development server
- `npm run android` - Start the app on Android
- `npm run ios` - Start the app on iOS
- `npm run web` - Start the app in web browser
- `npm run lint` - Run ESLint
- `npm run reset-project` - Reset the project configuration
